/*
** Format Specifiers:
**   ft_printf supports the following format specifiers:
**   - %c: Character
**   - %s: String
**   - %d or %i: Integer
**   - %u: Unsigned Integer
**   - %x: Lowercase Hexadecimal
**   - %X: Uppercase Hexadecimal
**   - %p: Pointer
**   - %%: Percent Sign
*/
